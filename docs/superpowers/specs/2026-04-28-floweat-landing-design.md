# FLOWEAT Landing Page – Design Spec
**תאריך:** 2026-04-28  
**סטטוס:** מאושר

---

## מטרה
דף נחיתה חדש לתוכנית FLOWEAT של שנהב בנימין. מטרה: המרת גולשות ללידים (פנייה בווטסאפ). קהל: נשים 30-60 שעייפות מדיאטות.

---

## Design System

### צבעים
| תפקיד | ערך | שימוש |
|---|---|---|
| רקע ראשי | `#FAF6F1` | רוב הסקשנים |
| רקע כהה | `#1C1008` | סקשנים מנוגדים (Hero, Pain, Testimonials) |
| טקסט ראשי | `#1A0D07` | כל הטקסט על רקע בהיר |
| אקצנט נחושת | `#C4704A` | הדגשות, קווים, כפתורים |
| אקצנט סאז' | `#8FA888` | אלמנטים משניים, check marks |
| כרטיסיות | `#FFFFFF` | cards עם צל חם |
| טקסט על כהה | `#F5EDE4` | על רקע `#1C1008` |

### טיפוגרפיה
- פונט: **Heebo** (300/400/700/900)
- Hero headline: `clamp(2.8rem, 9vw, 5.5rem)` weight 900
- Section headline: `clamp(1.8rem, 4.5vw, 2.8rem)` weight 700
- גוף: 17-18px, line-height 1.9
- כיוון: RTL לכל הדף

### אלמנטים חזותיים
- קווי-פסים: נחושת `#C4704A`, opacity 0.35
- צללים: `0 4px 24px rgba(28,16,8,0.08)`
- border-radius: 20px לכרטיסיות, 9999px לכפתורים
- אייקונים: ✦ ✓ ← (ללא emojis כבדים)
- אנימציות: Framer Motion, fade+slide, stagger 0.08s בין אלמנטים
