# אתר שנהב בנימין – תזונה מנטלית

**תאריך:** 2026-05-03  
**סטטוס:** מאושר

---

## סקירה כללית

אתר ראשי חדש נפרד מדף הנחיתה הקיים (floweat-landing). האתר משמש "בית" למותג עם ספריית מתכונים, תוכניות דיגיטליות, וקישור לדף הנחיתה של FLOWEAT.

---

## ארכיטקטורה

- **פרמוורק:** Next.js 15, App Router
- **עיצוב:** Tailwind CSS + Framer Motion
- **נתונים:** קבצי JSON סטטיים (`data/recipes.json`, `data/programs.json`)
- **רינדור:** Static Site Generation – כל עמוד נבנה בזמן build
- **כיוון:** RTL עברית

### מבנה נתוני מתכון

```json
{
  "slug": "string",
  "title": "string",
  "category": "סלטים | קינוחים | חלבון | ארוחות בוקר | מרקים | אחר",
  "ingredients": ["string"],
  "instructions": ["string"],
  "instagram_url": "string",
  "image_url": "string | null"
}
```

### מבנה נתוני תוכנית

```json
{
  "slug": "string",
  "title": "string",
  "description": "string",
  "price": "string",
  "landing_url": "string",
  "active": true
}
```

## עמודים

### `/` – דף הבית
- Hero: תמונה של שנהב + משפט אחד חזק
- שני בלוקים: "מתכונים" ו"תוכניות" עם קישור לכל אחד
- קישור לדף הנחיתה של FLOWEAT

### `/recipes` – ספריית מתכונים
- כפתורי סינון: הכל / סלטים / קינוחים / חלבון / ארוחות בוקר / מרקים
- גריד כרטיסיות: תמונה, שם, תגית קטגוריה
- 3 עמודות בדסקטופ, 2 בטאבלט, 1 במובייל

### `/recipes/[slug]` – עמוד מתכון
- שם המתכון, קטגוריה
- רשימת מצרכים
- הוראות הכנה (ממוספרות)
- כפתור "לסרטון באינסטגרם" – פותח בטאב חדש

### `/programs` – תוכניות
- כרטיסייה לכל תוכנית עם תיאור + מחיר + כפתור לדף הנחיתה
- כרגע: FLOWEAT בלבד
- עיצוב שמשאיר מקום לתוכניות עתידיות

## עיצוב

### פלטת צבעים (אדמה – ירוק וחום)

| שם | Hex | שימוש |
|---|---|---|
| ירוק עמוק | `#4A6741` | כפתורים ראשיים, לינקים, אקסנטים |
| ירוק sage | `#8FAF87` | תגיות קטגוריה, hover רקעים |
| חום אגוז | `#A67C52` | כפתורים משניים, border פעיל |
| חום חול | `#D4B896` | גבולות כרטיסיות, קווי הפרדה |
| רקע | `#FAFAF7` | רקע כללי |
| טקסט | `#1A1A1A` | טקסט ראשי |
| טקסט משני | `#6B6B6B` | כתוביות, metadata |

### טיפוגרפיה
- **פונט:** Heebo (Google Fonts) – עברית מודרנית וקריאה
- **כותרות:** Bold 700
- **גוף:** Regular 400
- **גדלים:** `text-4xl` לכותרות ראשיות, `text-xl` לכותרות משנה, `text-base` לגוף

### אנימציות
- כרטיסיות מתכון: scale קל + shadow בעת hover (Framer Motion)
- כניסה לעמוד: fade-in עדין
- סינון קטגוריות: transition חלק בין מצבים
